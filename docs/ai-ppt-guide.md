## 🚀 方法一：一键生成PPT（最快，5分钟搞定）

以 **Gamma** 为例：

### 第1步：打开网站
打开 [gamma.app](https://gamma.app)，用邮箱或Google账号注册登录。

### 第2步：新建项目
- 点击「新建」按钮
- 选择「生成」模式

### 第3步：输入主题
在输入框里写清楚你要做什么PPT，比如：

```
请帮我做一份关于"2026年公司上半年工作总结"的PPT，包括：
1. 上半年业绩回顾
2. 重点项目进展
3. 团队建设情况
4. 下半年工作计划
风格要求：商务正式，蓝色系
```

### 第4步：选择风格
- AI会提供几种模板风格供你选择
- 点击你喜欢的风格

### 第5步：等待生成
- AI会自动生成完整的PPT（大约30秒-1分钟）
- 包含标题、内容、配图、排版

### 第6步：微调修改
- 点击任意文字可以直接编辑
- 可以拖拽调整图片位置
- 可以更换整体配色方案

### 第7步：导出下载
- 点击右上角「导出」按钮
- 选择PPT格式（.pptx）下载
- 也可以导出PDF格式

## 🔧 方法二：大纲转PPT（效果更可控）

以 **MindShow** 为例：

### 第1步：先准备大纲
在任意AI工具（如Kimi、豆包）中输入：

```
请帮我写一份PPT大纲，主题是"如何培养阅读习惯"，要求：
- 10-15页
- 每页有标题和3-4个要点
- 语言简洁，适合演示
```

### 第2步：复制大纲到MindShow
1. 打开 [mindshow.fun](https://www.mindshow.fun)
2. 注册登录
3. 点击「导入创建」
4. 把AI生成的大纲粘贴进去

### 第3步：选择模板
- 浏览模板库，选择喜欢的风格
- 点击应用

### 第4步：预览和调整
- 预览每一页的效果
- 调整不满意的文字或排版
- 可以添加/删除页面

### 第5步：导出
- 点击「导出」下载PPT文件

## 💡 让PPT更好看的5个技巧

### 1. 给AI更详细的指令

❌ 简单指令：
```
帮我做一份关于营销的PPT
```

✅ 详细指令：
```
帮我做一份"2026年Q1营销策略方案"PPT：
- 共12页
- 受众：公司管理层
- 风格：简洁商务，深蓝色主题
- 包含数据图表页
- 每页文字不超过30字
```

### 2. 控制每页文字量

AI做PPT最常见的问题是每页文字太多。可以这样要求：

```
请确保每页PPT的内容简洁：
- 标题不超过10个字
- 正文不超过3个要点
- 每个要点不超过15个字
```

### 3. 分步生成内容页

先生成整体框架，再逐页优化：

```
我觉得第3页"市场分析"的内容太简单了，请帮我丰富一下，
加一些具体数据和趋势分析。
```

### 4. 善用AI配图

```
请为每一页PPT配上合适的图片：
- 风格统一，扁平化插画风格
- 不要用真人照片
- 配色和PPT主题一致
```

### 5. 加上动画效果

```
请给这份PPT添加适当的动画效果：
- 标题用飞入效果
- 要点逐条出现
- 切换页面用淡入淡出
```

## ⚠️ 注意事项

1. **检查内容准确性**：AI可能会编造数据，务必核实
2. **调整排版**：AI生成的排版不一定完美，手动微调更好
3. **注意图片版权**：AI生成的图片一般可以免费使用，但要确认
4. **保存源文件**：修改好后记得保存，方便下次修改
5. **适当留白**：PPT不要塞太多内容，简洁才有力量
