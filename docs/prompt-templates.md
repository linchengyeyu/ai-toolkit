## 一、✍️ 写文章

### 模板1：写公众号推文

```
你是一位有5年经验的新媒体运营专家。请帮我写一篇微信公众号推文：

- 主题：【填写主题，如：如何利用碎片时间学习】
- 目标读者：【填写，如：25-35岁的上班族】
- 字数：【填写，如：1500字】
- 风格：轻松有趣，像和朋友聊天，多用比喻和类比
- 结构：开头用一个能引起共鸣的场景引入 → 中间分3-4个实用建议 → 结尾有金句总结
- 要求：段落短小，每段不超过3行，适合手机阅读

请先给我3个不同的标题选项，选好后再写正文。
```

### 模板22：持续追问模板

当AI的回答不够好时，可以用这些话继续追问：

```
- "请说得更具体一些，给出实际的例子"
- "请用更简单的话重新解释一遍"
- "这个回答太长了，请精简到原来的1/3"
- "请从另一个角度来分析这个问题"
- "请列出更多选项，至少给我5个选择"
- "请把上面的内容整理成表格"
```

## 📌 提示词编写黄金法则

1. **角色设定**：告诉AI它是谁（如"你是一位资深文案"）
2. **背景信息**：提供足够的上下文
3. **明确要求**：说清楚格式、字数、风格
4. **给出示例**：如果有参考，发给AI看看
5. **分步完成**：复杂任务拆成小步骤，一步步来
6. **反复迭代**：第一次的回答不满意就继续追问
